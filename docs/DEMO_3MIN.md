# 3-Minute Demo Script - Agentic Decision Intelligence

**Pace:** Fast, high-energy. No pauses. Pre-load all tabs.

---

## PRE-DEMO SETUP (do before recording)

1. Start app: `mvn spring-boot:run`
2. Open http://localhost:8080 in Chrome at 110% zoom
3. Pre-click each tab once so data is cached (especially Briefing — takes 10s)
4. Return to RCA Chat tab (landing page)

## VIDEO TIPS

- **Recording:** Use Loom (easiest) or QuickTime Screen Recording
- **Speed up** loading screens to 2x in post
- **Pre-cache everything** so transitions are instant
- **Browser:** Full screen, no bookmarks bar, 110% zoom
- **Voice:** Confident, fast pace, no filler words
- **Total with loading pauses:** Record ~4 min, speed-edit to 3 min
